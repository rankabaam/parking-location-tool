# Workflow

This document describes the general workflow concept behind the Parking Location Tool.

## Purpose

The Parking Location Tool is designed to help dispatchers quickly identify aircraft parking locations from fuel request information.

In a busy aviation operations environment, parking information may come from emails, verbal communication, or dispatcher notes. This project demonstrates how that information can be organized into a simple structured format for faster lookup.

## General Workflow

```text
Fuel Request Email
        ↓
Email Folder / Inbox
        ↓
Data Extraction
        ↓
Structured Table
        ↓
Dashboard View
        ↓
